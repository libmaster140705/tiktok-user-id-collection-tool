## Key Features

### TikTok UID Collection

MadHub's **TikTok UID Collection** feature allows you to gather user IDs from various sources on TikTok, specifically targeting blogger followers and followings. This feature supports three types of collection: blogger's followers, blogger's followings, and the direct processing of a specified profile. With advanced filtering options, you can precisely target the users that match your marketing criteria.

#### Capabilities:
- **Multi-dimensional Social Data Filtering**: Filter users based on several criteria such as follower count, activity level, and other relevant metrics to ensure you are reaching your ideal audience.
- **Batch Operations Support**: Perform bulk actions such as follows or messages on the collected user IDs smoothly and efficiently.

#### Usage:
To utilize the **TikTok UID Collection** feature, simply configure the following steps:

1. **Select Collection Type**: Choose between collecting followers, followings, or direct profiles.
2. **Apply Filters**: Specify your filtering criteria such as minimum follower count or engagement metrics.
3. **Initiate Collection**: Start the collection process and watch as MadHub gathers the necessary data for you.

#### Configuration Example:
```python
# Configuration settings for TikTok UID Collection
set_collection_type('blogger_followers')  # Options: blogger_followers, blogger_following, profile
set_filter_conditions({'min_followers': 1000, 'max_followers': 10000})  # Adjust based on your targeting criteria
start_collection()
```

### TikTok User Search

The **TikTok User Search** feature in MadHub allows you to automatically search for TikTok users based on various keywords. This feature is instrumental for marketers and businesses looking to connect with potential clients or influencers in their niche.

#### Capabilities:
- **AI-Enhanced Filtering**: Utilize AI to filter users based on specific criteria such as follower count and engagement levels, ensuring you reach highly relevant accounts.
- **Interactivity**: Perform actions like following or sending direct messages to potential leads directly from the user search results.

#### Usage:
To leverage the **TikTok User Search** functionality effectively:

1. **Input Keywords**: Enter relevant keywords to initiate your search.
2. **Set Filters**: Apply any desired filters such as follower count or post frequency.
3. **Execute Search**: Run the search and MadHub will present a list of potential users that match your criteria.

#### Configuration Example:
```python
# Configuration settings for TikTok User Search
set_search_keywords(['influencer', 'marketing tips'])  # Define keywords for searching
